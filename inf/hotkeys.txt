                   Горячие клавиши редактора:


                     Функциональные клавиши:

< F1 Описание > - Показать описание символа или метки

< F2 Файл > - Открыть файл где описана символ или метка, для директивы .include - открыть файл указанный в директиве

< F3 Определение > - Перейти к определению символа или метки, в случае если символ или метка определены в другом файле - он так же будет открыт

< F4 Изменить > - Для символов: в зависимости от директив @.item и @.config будут выбраны значения из одной группы символов. Если директивами реакция задана не будет то список выбора будет предложен по @.item уже определенного символа (который изменяем), если @.item символа не задан - то реакции не будет
Для директивы .include - выбор файла, при вставке имени файла его путь будет усечен до пути к исходным файлам проекта

< F5 Символы > - Открывает окно выбора символа или метки для вставки


                    Сочетания с управляющими клавишами:

Работа с буфером обмена:
------------------------
< Ctrl + A >  - выделить весь текст в окне редактора

< Ctrl + C >  - копировать выделение в буфер обмена

< Ctrl + X >  - вырезать выделенный текст в буфер обмена

< Ctrl + V >  - вставить текст из буфера обмена

Форматирование текста:
----------------------
< Ctrl + D >  - форматирование строки или группы строк по выделению

< Ctrl + Y >  - удалить строку в которой стоит курсор

Поиск и замена:
---------------
< Ctrl + F >  - диалог поиска строки в тексте

< Ctrl + R >  - диалог замены строки в тексте

Операции с файлами:
-------------------
< Ctrl + O >  - открыть файл

< Ctrl + N >  - Создать новое окно редактора файла

< Ctrl + S >  - Сохранить текст текущего окна редактора

< Shift + Ctrl + S > - Сохранить текст окна редактора с заданием нового имени

Управление изменениями в тексте:
--------------------------------
< Ctrl + Z >  - отмена последнего изменения текста

< Ctrl + U >  - возврат отмененных изменений


Прочие:
-------
< Сtrl + Alt  + Курсор влево/вправо > - Сдвиг токена строки влево/вправо

< Сtrl + Курсор влево/вправо> - Перемещение по токенам строки влево/вправо

< Сtrl + Курсор вверх/вниз> - смещение окна просмотра текста без перемещения курсора

< Сtrl + '+' > - Увеличить размер шрифта в редакторе

< Сtrl + '-' > - Уменьшить размер шрифта в редакторе
